# odoo-react-rpc-integration

A full-stack project demonstrating integration between **Odoo** and **React** using **RPC calls**.  
The backend is built with **Node.js** to communicate with Odoo models running on localhost, and the frontend is developed in **React** to display model data and columns dynamically.
